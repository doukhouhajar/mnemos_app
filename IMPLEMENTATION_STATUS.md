# MNEMOS Implementation Status

## Current Functionality

MNEMOS is a calendar-native spaced repetition learning app built with React Native and Node.js that helps users capture, structure, and review learning content using science-based memory techniques. The app currently features a fully functional calendar interface that serves as the primary navigation hub, displaying scheduled memory reviews with visual indicators for different activity types (Review, Challenge, Reflection). Users can capture learning moments through a floating action button that opens a bottom sheet interface, with options for manual input or AI-assisted structuring via OpenAI integration (when API key is configured). The calendar supports date-specific views where clicking any day reveals all learning moments captured on that date, along with their associated memory objects. The review system implements the SM-2 spaced repetition algorithm, automatically calculating optimal review intervals based on user performance, confidence scores, and response latency. The app includes six different experience types (Free Recall, Cued Recall, Application, Explain Simply, Misconception Detection, and Micro Teach) that users can select before engaging with memory reviews, each designed to test different aspects of retention and understanding. Additionally, the app features interactive Groups and Weekly Quests functionality, where users can view detailed participant lists, shared memories, join or leave groups, and participate in collaborative learning challenges with progress tracking. The UI follows a minimal, Cursor-inspired design philosophy with simple unicolor icons, soft typography, and a calm color palette that emphasizes cognitive clarity over gamification. All core features are connected to a PostgreSQL backend with a complete RESTful API, including endpoints for memory objects, learning moments, reviews, schedule states, and AI-assisted content generation.
